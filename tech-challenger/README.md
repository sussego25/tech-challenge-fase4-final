# Tech Challenger

Arquitetura de processamento assíncrono com SQS, EKS, SageMaker e DynamoDB.

## Estrutura do Projeto

- **docs/**: Documentação da arquitetura e diagramas
- **infra/**: Configurações de infraestrutura (Terraform/CDK)
- **services/**: Serviços da aplicação
- **messaging/**: Documentação das filas SQS
- **shared/**: Código compartilhado e contratos
- **deploy/**: Configurações de deploy (Docker, Kubernetes, Helm)
- **tests/**: Testes de integração e E2E
